软件简介
拍书题库 是一款 AI 驱动的智能题库工具。

教师只需拍一张课本照片或上传 PDF / Word 文档，AI 即可自动提取内容并生成题库，并可以使用题库生成试卷，还支持教师间相互分享题库。

学生端（同一WIFI下或局域网中）通过扫码或网址，即可完成在线练习和考试，教师无需部署服务器，学生也无需安装任何软件，做题和考试同时还支持离线使用，在无网络的手机或电脑中也可做题或考试。
我的角色
🎓教师 / 出题者
📖学生 / 答题者
🎓 教学端 · 5 步快速上手
1
进入 「设置」 → 「AI 配置」，填写 API Key 和模型名称（首次必做）
2
进入 「题库」，点击右下角 ＋，上传课本 PDF / 图片或直接拍照
3
填写题库名称、科目、题目数量，点击 「开始生成」，等待 AI 出题
4
进入 「设置」 → 开启本地服务器，把二维码或链接分享给学生
5
在 「考试」 页发布指定考试或 抢答比赛；在 「报告」 页查看成绩和排名
📖 学生端 · 5 步快速上手
1
从老师处获取 网址或二维码，用手机 / 电脑浏览器打开
2
点击 「注册」，填写真实姓名、学号、班级，设置密码
3
登录后，若老师已发布考试会 自动跳转到指定考试；否则自由选题库练习
4
练习答错的题自动进入 错题本，可在「错题重练」中专项复习
5
考试结束后查看 成绩单和排名，每题有答案解析供复习
💡 教师提示：含数学公式或图表的课本，新建时请选「图表公式模式」，识别更准确。
✅ 学生提示：无需安装 App，用浏览器扫码即可在线答题，支持练习和考试两种模式。
第一步：配置 AI (用于使用AI创建题库)
1
进入底部 「设置」 → 「AI 配置」卡片
2
填写 API Key（支持 OpenAI、DeepSeek、通义千问等兼容接口，自行在相应平台购买或联系作者购买）
3
填写 Base URL（使用国内中转或非 OpenAI 服务时必填）
4
填写 文本模型 名称（出题用）和 视觉模型 名称（图表公式模式用）
5
填写 系统提示词（可选）：自定义出题风格，如「只出高考难度，计算题占 30%」
题库管理
📖
新建题库
上传文档或拍照，AI 自动生成题目
›
  • 点击「题库」页右下角 ＋ 按钮
  • 支持文件：PDF、Word（.doc/.docx）、TXT、Markdown、图片（JPG/PNG 等）
  • 可同时选择 多个文件，AI 合并识别后统一出题
  • 也可直接 拍照，适合纸质课本和白板内容
  • 或在其他 App 中把文件 分享给拍书题库，自动触发出题流程
  • 填写题库名称（必填）、科目、出题数量、附加要求（可选）
  • 普通文本模式：先 OCR 识别文字再生成，速度快，适合文字为主的教材
  • 图表公式模式：图片直接发给 AI 视觉模型，精准识别数学公式、表格、图示
➕
追加题目
向已有题库继续添加新题
›

  • 分享文件到拍书题库时，也可选择「添加到已有题库」

  • 点击 删除 图标单题删除
  • 可设置题目 难度（⭐1-5） 和 知识点标签，用于考试筛选

发布在线指定考试
为指定学生发布强制考试任务
›

  • 选择题库，设置各题型 数量 和 每题分值（自动计算总分）

  • 设置 可交卷时间：距结束 X 分钟前允许手动交卷，0 = 不允许提前交卷

  • 设置 开考时间：立即 / 定时（选日期和时间）
  • 勾选参与学生（支持全选），点击「发布 → 在线考试」

抢答比赛
实时竞速答题，大屏展示排名
›
  • 「考试」→「发布考试」→ 选择题库和题数，点击「发布 → 抢答比赛」
  • 学生登录后自动进入 抢答等待界面，无需手动操作
  • 教师在浏览器中打开 手机IP:8081（或点击「统计台」链接）进入大屏统计台
  • 统计台中选择比赛 → 点击 「开始比赛」，再逐题点击「出题」向所有学生同步题目

  • 每题结束后点击「下一题」继续；所有题目完成后点击「结束比赛」显示总结页
  • 总结页展示最难题目和学生/小组最终排名；支持 「重新比赛」 一键重置后再次开始
  • 学生分组：进入「学生管理」为学生设置组别，同组成员成绩合并统计

题库 & AI 生成
Q生成题目失败，怎么排查？›
检查「设置 → AI 配置」中 API Key 是否正确；如使用国内中转，确认 Base URL 无误；确认手机能正常访问 AI 服务。可开启「调试模式」查看详细错误。
QPDF 内容识别不全或乱码？›
新建题库时切换为「图表公式模式」，AI 直接读取 PDF 图片，无需 OCR，识别更完整。扫描版 PDF 尤其推荐此模式。
Q生成的题目质量不理想？›
在「设置 → AI 配置 → 系统提示词」添加具体要求，如「出高考难度，覆盖所有知识点，计算题占 30%」。也可切换更强大的模型（GPT-4o、deepseek-chat 等）。
Q如何生成更多题目？›
新建题库时增大「题目数量」；或生成后点「继续添加题目」，上传更多文件或补充提示词，多次追加。
服务器 & 网络
Q抢答统计台（大屏）如何打开？›
发布抢答比赛后，在浏览器中访问 手机IP:8081，或点击「考试」列表中的「统计台」链接。建议在大屏/电视或教室电脑上打开，效果最佳。
Q抢答比赛学生看不到题目？›
确认教师已在统计台点击了 「开始比赛」 并且点击了 「出题」。题目只有在教师主动出题后才会同步给学生；学生登录后需等待教师操作。
Q学生无法访问服务器？›
确认师生设备在 同一 WiFi。若教师开了手机热点，学生连热点后可访问。仍不通则在「设置」页重启服务器，或检查手机防火墙 / 安全软件是否拦截端口。
Q如何让学生在家远程访问？›
内嵌服务器仅支持局域网。如需远程访问，可使用 内网穿透（frp、ngrok、花生壳等）将本地端口暴露到公网。
Q关掉 App 后学生还能访问吗？›
服务器作为 前台服务 后台持续运行，锁屏后仍可访问。建议将本应用设置为「不限制后台」并关闭省电优化，防止系统强制杀掉服务。

Q如何备份题库数据？›
长按题库卡片 → 「导出题库」，将 .sqb 文件保存到网盘。换手机后打开该文件即可恢复。
Q学生忘记密码怎么重置？›
教师进入「学生管理」→ 点击该学生 编辑 → 密码栏填写新密码 → 保存。学生无法自助重置，需教师操作。
Q清空考试记录会影响题库吗？›
不会。「清空考试记录」只删除历史答题 / 考试数据，题目本身不受影响。错题记录也会一并清除。
